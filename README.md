# react-redux-route-sass-login
This project is prepared template for login logout with react redux route sass

## How to use

First copy the repo into your disk.

```bash
$ git clone  https://github.com/bsmahala/react-redux-route-sass-login.git
```
# Install dependancies
  cd react-redux-route-sass-login
  
  npm install
  
  npm start
# Production Build
  npm run build
